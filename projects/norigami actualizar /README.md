# Norigami - Sushi & Tortas Ahogadas

Sitio web para Norigami, restaurante de fusión japonesa y tortas ahogadas en Guadalajara.

## 📁 Estructura

```
norigami actualizar /
├── index.html          # Página principal
├── norigami-logo.png   # Logo del restaurante
├── el-toro_logo.png    # Logo El Toro
├── hero-sushi.jpg      # Imagen hero principal
├── cat-*.jpg           # Imágenes de categorías del menú
├── gallery-*.jpg       # Imágenes de galería
└── ...
```

## 🚀 Despliegue

Los archivos están listos para ser desplegados en un servidor web estático.

### Opción 1: GitHub Pages
1. Configurar GitHub Pages en la rama `master`
2. El sitio estará disponible en `https://0ariel02.github.io/reutiliza_openclaw/norigami%20actualizar%20/`

### Opción 2: Servidor local
```bash
cd "norigami actualizar "
python3 -m http.server 8000
# Acceder a http://localhost:8000
```

### Opción 3: Netlify / Vercel
- Arrastrar la carpeta `norigami actualizar ` a Netlify Drop
- O conectar el repositorio para despliegue automático

## 🎨 Características

- Diseño responsive
- Tema claro/oscuro
- Navegación suave
- Galería de imágenes
- Menú completo con categorías
